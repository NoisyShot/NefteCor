# Проект Go-scooter-pass

**Описание проекта**

- Адаптивный одностраничный лэндинг

**Использованные технологии**

- HTML, CSS, JavaScript.
- Flexbox-верстка, Grid-Layout.
- Медиазапросы и оптимизация под разрешения 1440px и 425px.
- Desktop first
- Адаптивная верстка.
